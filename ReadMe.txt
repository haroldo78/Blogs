ReadMe Blogs

These are projects that were created for companies to share content with their clients, strengthening their networking, covering a multitude of subjects that are posted daily or weekly, updating their clients quickly and with true content.

Using: HTML5, CSS3, Sass, Grunt, Joomla Content Management System with responsive design. With components: Jquery, analytics.js GoogleAnalyticsObject, Media querie responsive.

Can be used with a local served xampp, mamp or of your preference. given start in Apache and MySQL. to create a database of name blogoconsultor in admin, phpmyadmin.

The configuration.php
put the folder path and put the name of the database
public $ db = 'bloglci';

public $ log_path = 'C:\\xampp\\htdocs\\blog-lci\\site/logs';
public $ tmp_path = 'C:\\xampp\\htdocs\\blog-lci\\site/tmp';

put the folder path and put the name of the database

public $ log_path = 'C:\\xampp\\htdocs\\blog-papoconsultor\\site/logs';
